# APK编译指南

本指南说明如何将HTML+JS项目编译成Android APK安装包。

## 快速开始（最简单）

### 1. 使用在线编译平台（无需安装环境）

推荐平台：[Capacitor Cloud](https://capacitorjs.com/docs/basics/building-your-app)

步骤：
1. 注册并登录平台
2. 上传代码包
3. 选择Android平台
4. 点击编译，下载APK

### 2. 使用GitHub Actions自动化编译

项目已包含GitHub Actions配置，推送到GitHub即可自动编译APK。

步骤：
1. 将项目推送到GitHub仓库
2. 进入仓库的"Actions"页面
3. 等待编译完成
4. 在Artifacts中下载APK

## 本地编译（推荐开发者）

### 环境准备

#### 安装Node.js
```bash
# macOS
brew install node

# 验证版本（需要16+）
node --version
npm --version
```

#### 安装JDK 11
```bash
# macOS
brew install openjdk@11

# 设置环境变量
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v11)' >> ~/.zshrc
source ~/.zshrc

# 验证
java -version
javac -version
```

#### 安装Android Studio
1. 下载并安装：https://developer.android.com/studio
2. 打开Android Studio
3. SDK Manager → 安装 Android SDK Platform 34
4. 安装 Android SDK Build-Tools 34.0.0

## 编译步骤

### 步骤1：安装依赖

```bash
cd /Users/leslie/WorkBuddy/20260319150702/kw_print_app
npm install
```

### 步骤2：初始化Capacitor

```bash
npm run cap:init
```

按提示输入：
```
✨ Creating a new Capacitor project at /Users/leslie/WorkBuddy/20260319150702/kw_print_app

? App 名字: 库位码打印
? App ID: com.kwprint.app
? Web 目录（dist, www 等）: .
```

### 步骤3：添加Android平台

```bash
npm run cap:add:android
```

### 步骤4：同步代码

```bash
npm run cap:sync
```

### 步骤5：打开Android Studio

```bash
npm run cap:open:android
```

Android Studio会自动打开项目。

### 步骤6：在Android Studio中编译

#### 方法A：通过菜单编译
1. 菜单栏：`Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
2. 等待编译完成（约1-3分钟）
3. 点击通知中的"locate"找到APK文件
4. APK路径：`app/build/outputs/apk/debug/app-debug.apk`

#### 方法B：通过命令行编译
在Android Studio的Terminal中执行：
```bash
./gradlew assembleDebug
```

#### 方法C：生成签名的APK（发布用）
1. `Build` → `Generate Signed Bundle / APK`
2. 选择`APK`
3. 创建或选择Keystore
4. 选择`debug`或`release`版本
5. 点击完成

## 快速编译脚本

创建一键编译脚本：

```bash
#!/bin/bash
# build_apk.sh

echo "🚀 开始编译APK..."

# 清理旧文件
rm -rf app/build/outputs/apk/*

# 编译Debug APK
./gradlew clean
./gradlew assembleDebug

# 复制APK到项目根目录
cp app/build/outputs/apk/debug/app-debug.apk kw-print-app.apk

echo "✅ APK编译完成：kw-print-app.apk"
```

使用方法：
```bash
cd android
chmod +x build_apk.sh
./build_apk.sh
```

## 常见编译问题

### 问题1：Gradle下载失败

解决方法：
```bash
# 使用国内镜像
编辑 ~/.gradle/init.gradle

allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/jcenter' }
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        jcenter()
    }
}
```

### 问题2：JDK版本不匹配

解决方法：
```bash
# 查看当前Java版本
java -version

# 如果不是JDK 11，切换版本
/usr/libexec/java_home -v 11
```

### 问题3：Android SDK未安装

解决方法：
1. 打开Android Studio
2. SDK Manager → 安装SDK Platform 34
3. 安装Build-Tools 34.0.0

### 问题4：编译失败：BUILD FAILED

查看详细错误：
```bash
./gradlew assembleDebug --stacktrace
```

## 版本管理

### 修改版本号

编辑 `android/app/build.gradle`：

```gradle
android {
    defaultConfig {
        applicationId "com.kwprint.app"
        minSdkVersion 24
        targetSdkVersion 34
        versionCode 2  // 递增
        versionName "1.0.1"
    }
}
```

### 生成版本信息
```bash
# 格式：kw-print-app-v1.0.0-$(date +%Y%m%d).apk
cp app/build/outputs/apk/debug/app-debug.apk "kw-print-app-v1.0.0-$(date +%Y%m%d).apk"
```

## 进阶配置

### 配置应用图标

1. 准备图标文件（512x512 PNG）
2. 访问：https://icon.kitchen/
3. 上传图标，下载资源包
4. 解压并复制到 `android/app/src/main/res/`

编辑 `android/app/src/main/res/values/strings.xml`：

```xml
<resources>
    <string name="app_name">库位码打印</string>
</resources>
```

编辑 `android/app/src/main/AndroidManifest.xml`：

```xml
<uses-permission android:name="android.permission.BLUETOOTH" />
<uses-permission android:name="android.permission.BLUETOOTH_ADMIN" />
```

## 提交到应用商店

### 生成签名的APK

1. 创建Keystore：
```bash
keytool -genkey -v -keystore kw-print.keystore -alias kwprint -keyalg RSA -keysize 2048 -validity 10000
```

2. 配置签名：
编辑 `android/app/build.gradle`：

```gradle
android {
    signingConfigs {
        release {
            storeFile file("kw-print.keystore")
            storePassword "你的密码"
            keyAlias "kwprint"
            keyPassword "你的密码"
        }
    }

    buildTypes {
        release {
            signingConfig signingConfigs.release
            minifyEnabled false
            proguardFiles getDefaultProguardFile('proguard-android.txt'), 'proguard-rules.pro'
        }
    }
}
```

3. 编译Release APK：
```bash
./gradlew assembleRelease
```
