# 一键构建APK - 使用说明

## 🚀 构建APK（只需一条命令）

```bash
cd /Users/leslie/WorkBuddy/20260319150702/kw_print_app
./build_apk.sh
```

---

## 📋 构建流程

脚本会自动完成以下步骤：

1. ✅ 检查构建环境（Node.js、JDK、Android SDK）
2. ✅ 安装 npm 依赖
3. ✅ 配置 Capacitor 项目
4. ✅ 添加 Android 平台
5. ✅ 同步代码到 Android 项目
6. ✅ 编译 APK
7. ✅ 复制到项目根目录：`kw-print-app.apk`

---

## ⚙️ 环境要求

### 必需组件

| 组件 | 版本要求 | 安装命令 |
|------|---------|---------|
| Node.js | 16+ | `brew install node` |
| JDK | 11+ | `brew install openjdk@11` |
| Android SDK | 最新版 | 安装 Android Studio |
| Android Build Tools | 34.0.0 | 在 Android Studio 的 SDK Manager 中安装 |

### 检查环境

```bash
# 检查 Node.js
node -v

# 检查 JDK
java -version

# 检查 Android SDK（如果设置了环境变量）
echo $ANDROID_HOME
```

## 📱 安装到手机

### 方法一：自动安装（推荐）

脚本会自动检测是否连接了手机并尝试安装：

```bash
./build_apk.sh
# 按提示选择 y - 自动安装
```

### 方法二：手动安装

1. 将 `kw-print-app.apk` 复制到手机
2. 在手机上点击 APK 文件
3. 授予蓝牙、位置等权限
4. 安装完成

### 方法三：通过 ADB 安装

```bash
# 连接手机并启用USB调试
adb devices

# 安装APK
adb install kw-print-app.apk
```

## 🔧 常见问题

### Q1: 脚本执行报错：Node.js 未安装

**解决方法：**
```bash
brew install node
```

### Q2: 脚本执行报错：JDK 未安装

**解决方法：**
```bash
brew install openjdk@11
```

### Q3: 脚本执行报错：未找到 Android SDK

**解决方法1：** 安装 Android Studio
1. 下载：https://developer.android.com/studio
2. 安装后打开 SDK Manager
3. 安装 Android SDK Platform 34
4. 安装 Build-Tools 34.0.0

**解决方法2：** 设置环境变量
```bash
# 添加到 ~/.zshrc 或 ~/.bash_profile
export ANDROID_HOME="$HOME/Library/Android/sdk"
export PATH="$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools:$PATH"

# 重新加载配置
source ~/.zshrc
```

### Q4: Gradle 下载失败或很慢

**解决方法：** 使用国内镜像

编辑 `~/.gradle/init.gradle`（创建文件）：
```gradle
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/jcenter' }
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        jcenter()
    }
}
```

### Q5: 构建失败：BUILD FAILED

**解决方法：**
```bash
# 查看详细错误信息
cd android
./gradlew assembleDebug --stacktrace
```

### Q6: 脚本权限不足

**解决方法：**
```bash
chmod +x build_apk.sh
./build_apk.sh
```

## 🚨 重要提示

1. **首次构建**：Gradle 会下载依赖，需要较长时间（5-10分钟）
2. **后续构建**：依赖已缓存，速度会快很多（2-3分钟）
3. **网络要求**：需要能访问互联网下载 Gradle 依赖
4. **权限要求**：脚本会请求打开文件夹、安装APK等操作
