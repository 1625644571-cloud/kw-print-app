# 在线编译APK - 快速指南

## 🚀 推荐方案：使用 Capacitor Cloud

官方在线编译平台，简单快速，无需本地配置环境。

---

## 方法1：Capacitor Cloud（最简单）

### 步骤1：准备代码

确保你的 `kw_print_app` 目录结构正确：

```
kw_print_app/
├── www/
│   ├── index.html
│   ├── app.js
│   ├── printer.js
│   └── printer_detong.js
├── capacitor.config.json
├── package.json
└── build_apk.sh
```

### 步骤2：访问 Capacitor Cloud

1. 打开浏览器访问：
   **https://capacitorjs.com/docs/basics/building-your-app**

2. 或直接使用 Ionic Appflow：
   **https://ionicframework.com/appflow**

### 步骤3：上传代码

**如果使用 Capacitor Cloud：**
1. 注册或登录账号
2. 创建新项目
3. 上传 `kw_print_app` 文件夹
4. 选择平台：Android
5. 点击 Build

**如果使用 Ionic Appflow：**
1. 注册账号（免费）
2. 点击 "New App"
3. 选择 "Import from Git"
4. 将代码推送到 GitHub 仓库
5. 在 Ionic Appflow 中连接仓库
6. 选择 Android 平台
7. 点击 Build

### 步骤4：下载 APK

构建完成后（约3-5分钟），点击下载 APK 文件。

## 方法2：使用 GitHub Actions（自动构建）

如果你熟悉 Git，这是最省事的方法。

### 步骤1：推送到 GitHub

```bash
# 初始化 Git
cd /Users/leslie/WorkBuddy/20260319150702/kw_print_app
git init
git add .
git commit -m "Initial commit"

# 创建 GitHub 仓库后
git remote add origin https://github.com/你的用户名/kw-print-app.git
git push -u origin main
```

### 步骤2：创建 GitHub Actions 配置

创建 `.github/workflows/build-apk.yml`：

```yaml
name: Build APK

on:
  push:
    branches: [ main, master ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v3

      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'

      - name: Install dependencies
        run: npm install

      - name: Setup Java
        uses: actions/setup-java@v3
        with:
          distribution: 'temurin'
          java-version: '11'

      - name: Setup Android SDK
        uses: android-actions/setup-android@v2

      - name: Initialize Capacitor
        run: |
          echo "y" | npx cap init "库位码打印" "com.kwprint.app" --web-dir=www

      - name: Add Android platform
        run: npx cap add android

      - name: Sync Android
        run: npx cap sync android

      - name: Build APK
        run: |
          cd android
          ./gradlew assembleDebug

      - name: Upload APK
        uses: actions/upload-artifact@v3
        with:
          name: kw-print-app
          path: android/app/build/outputs/apk/debug/app-debug.apk
```

### 步骤3：触发构建

推送代码后，GitHub 会自动构建APK。

### 步骤4：下载 APK

1. 进入 GitHub 仓库的 "Actions" 标签
2. 找到最新的构建记录
3. 在 "Artifacts" 中下载 `kw-print-app` 文件
