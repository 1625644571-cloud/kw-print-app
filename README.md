# 库位码自动打印APP

用于监听美团牵牛花APP，在确认收货时自动连接德佟P1蓝牙打印机打印库位码标签。

## 功能特点

✅ 后台监听牵牛花APP确认收货操作
✅ 自动连接德佟P1蓝牙打印机
✅ 支持ESC/POS打印指令
✅ 自动识别并打印库位码标签
✅ 实时状态监控和日志记录
✅ 支持测试打印功能

## 安装方法

### 方法一：直接安装APK（推荐）

1. 将打包好的 `kw-print-app-v1.0.0.apk` 复制到手机
2. 在手机上点击安装
3. 授予蓝牙、位置等权限
4. 打开APP即可使用

### 方法二：自己编译APK

#### 前置要求

- Node.js 16+
- JDK 11+
- Android Studio（可选）

#### 编译步骤

```bash
# 1. 安装依赖
npm install

# 2. 初始化Capacitor
npm run cap:init
# 按提示输入：
# App ID: com.kwprint.app
# App Name: 库位码打印
# Web Dir: .

# 3. 添加Android平台
npm run cap:add:android

# 4. 同步代码到Android项目
npm run cap:sync

# 5. 打开Android Studio
npm run cap:open:android

# 6. 在Android Studio中：
#   - 点击 Build → Build Bundle(s) / APK(s) → Build APK(s)
#   - 等待编译完成
#   - APK文件位于 app/build/outputs/apk/debug/
```

## 使用说明

### 首次使用

1. **打开APP**
   - 点击图标启动应用

2. **配置蓝牙打印机**
   - 系统自动扫描蓝牙设备
   - 找到"德佟P1"或"P1"设备
   - 点击设备名称选中
   - 点击"连接打印机"按钮

3. **测试打印**
   - 点击"测试打印"按钮
   - 检查打印机是否正常工作

4. **配置库位码匹配规则**（可选）
   - 默认规则：`^[A-Z]{2,3}-\d+-\d+$`（如 HZ-1-2）
   - 根据实际情况修改

5. **启动监听**
   - 点击"启动监听"按钮
   - 右下角出现悬浮球，显示"监听中"

### 日常使用

1. 打开APP，点击"启动监听"
2. 返回桌面，打开美团牵牛花APP
3. 进行正常的收货操作
4. 点击"确认收货"后，APP会自动：
   - 识别库位码
   - 连接蓝牙打印机
   - 打印库位码标签

## 状态说明

### 监听状态
- 🟢 监听中：正在后台监听牵牛花APP
- 🔴 已停止：监听功能已关闭

### 蓝牙打印机
- 🟢 已连接：打印机连接正常
- 🔴 离线：打印机未连接

### 牵牛花APP
- 🟢 正常：检测到APP正在运行
- 🔴 异常：未检测到APP

## 常见问题

### Q: 扫描不到蓝牙打印机？

A: 确保以下操作：
1. 打开手机蓝牙功能
2. 德佟P1打印机已开机并处于配对模式
3. 在系统蓝牙设置中先配对打印机
4. 关闭其他蓝牙应用

### Q: 打印失败？

A: 检查以下内容：
1. 打印机是否已连接
2. 打印机是否有纸
3. 打印机电池是否有电
4. 查看APP日志了解详细错误信息

### Q: 监听功能不工作？

A: 确认：
1. 牵牛花APP已在手机上安装
2. 已授予APP必要的权限（悬浮窗、无障碍等）
3. 牵牛花APP在前台运行

### Q: 库位码识别不准确？

A: 调整库位码匹配规则：
1. 在"配置卡片"中修改正则表达式
2. 根据实际库位码格式编写规则
3. 保存后重启监听

## 技术支持

- 打印机型号：德佟P1
- 蓝牙协议：BLE（低功耗蓝牙）
- 打印指令：ESC/POS
